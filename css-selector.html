CSS选择器简明

邓亮泉 前端

简略说一下文章的大纲 * 标签选择器 * 类选择器 * ID选择器 * 后代选择器 * 子元素选择器 * 相邻元素选择器 * 属性选择器 * 选择器的覆盖

标签选择器

h1{
	color: red;
}

<h1>我的文字将会变成红色</h1> 标签选择器就是让样式按照指定的标签去匹配需要应用这个样式的元素

类选择器

.error{
	color: red;
	padding: 15px;
  border: 1px #ccc red;
}

<div class=“error”>错误提示</div> 类选择器就是让样式按照class="样式"的方式去匹配需要应用这个样式的元素

ID选择器

#intro{
	line-height:1.5;
}

<p id="intro">简介... ba la ba ba la 如果你想看行高的效果的话，可以一直输入到换行，或者调整浏览器宽度至换行</p> 同上，顾名思义让样式按照id=“ID的方式”去匹配

后代选择器

nav li{
  background: red;
}

<ul class="nav">
  <li>Home</li>
  <li>Tags</li>
  <li>Abount</li>
  <li>User</li>
  <ul>
   <li>Settings</li>
   <li>Logout</li>
  </ul>
</ul>

通过以上示例可以看到nav下的子元素 li标签和其它子元素下的子元素li标签的背景色都被设置成了红色。

子元素选择器

nav>li{
  background: red;
}

<ul class="nav">
  <li>Home</li>
  <li>Tags</li>
  <li>Abount</li>
  <li>User</li>
  <ul>
   <li>Settings</li>
   <li>Logout</li>
  </ul>
</ul>

通过以上示例可以看到nav下的子元素 li标签的背景色应用了样式被设置成了红色，其它子元素下的li标签并没有应用样式。但是需要注意的是子元素选择器在IE7中有BUG，如果父元素和子元素之间有HTML注释的话，后面所有跟有注释的子选择器的选择器会被忽略，IE7的更高版本都是支持子元素选择器的，如果想在IE7或者IE8上使用类似于子选择器的效果可以参照以下示例

.nav li{
  background: red;
}

.nav li *{
  background: none;
}

相邻元素选择器

h1 + p {
  font-weight: bold;
}

<h1>文章标题</h1>
<p>提示</p>
<p>内容 ba la ba la</p>

通过以上示例我们可以看到相邻元素选择器只会对根据元素顺序下的下一个元素起作用，但是要注意IE，相邻元素选择器只在IE7(S)、IE8(S)或更高版本使用。

属性选择器

input[type="text"]{
  border: 1px solid #ccc;
}

a[data-href]{
	color: red;
}

<input type="text" name="username" >
<a data-href="https://www.google.com.hk/">跳转</a>

通过示例可以看到属性选择器只会对HTML代码中满足属性选择器的表达式的元素才会起作用

选择器的覆盖

.error{
	color: blue;
}
.error{
	color: red;
}

前者会被后者替换掉，接下来说一下选择器的优先级 style=“” > id > class >标签 选择器是可以一起使用的，那浏览器是如何计算采用哪种样式的？ style 1000分 id 1个100分 class 1个10分 标签 1个1分

#page{
  color: #0ebeff;
}
.container{
  color: #ccc;
}

<div id="page" class="container">
  我自年少,韶华倾覆。
</div>

结果显而易见id VSclass 100 > 1

#page h1{
  color: #0ebeff;
}
#page .title{
  color: red;
}

<div id="page" class="container">
  <h1 class="title">我自年少,韶华倾覆。</h1>
</div>

想一想会使用哪个样式？ 第一个样式 id+标签得分 101 第二个 id+class得分110
